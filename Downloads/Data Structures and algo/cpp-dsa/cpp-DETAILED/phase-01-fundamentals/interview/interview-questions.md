# Phase 1 - Interview Questions

## Frequently Asked Questions

### 1. Explain the C++ compilation process.

The compilation process converts C++ source code into an executable. The main stages are preprocessing, compilation, assembly, linking, and loading. Each stage has a distinct role, and understanding them helps identify whether an error is a syntax issue, a linker issue, or a runtime issue.

### 2. Why do we use constants?

Constants protect values that should not change during execution. They improve readability, prevent accidental modification, and help express intent.

### 3. What is the difference between scope and lifetime?

Scope is where a name can be accessed in code. Lifetime is how long the object exists in memory.

### 4. Why are functions important?

Functions improve modularity, reusability, readability, and testing. They also help organize logic into smaller pieces.

### 5. What is function overloading?

Function overloading allows multiple functions to share the same name as long as their parameter lists differ.
