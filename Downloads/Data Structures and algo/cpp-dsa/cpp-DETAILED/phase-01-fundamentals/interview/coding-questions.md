# Phase 1 - Coding Questions

## Easy

- Print a greeting message
- Add two numbers
- Find the largest of three numbers
- Check even or odd
- Calculate factorial using a loop
- Find the sum of digits
- Print a number pattern

## Medium

- Build a simple calculator using functions
- Convert temperature from Celsius to Fahrenheit
- Count vowels in a string
- Find prime numbers in a range
- Generate multiplication tables
- Print Pascal-like patterns

## Advanced

- Create a menu-driven utility program
- Handle multiple test cases from command line input
- Implement a modular calculator with separate functions
- Trace a program using assertions and debugging output
