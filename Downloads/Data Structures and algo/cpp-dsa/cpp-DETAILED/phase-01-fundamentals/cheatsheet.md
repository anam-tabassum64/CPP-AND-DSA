# Phase 1 - Cheat Sheet

## Syntax

```cpp
int main() {
    return 0;
}
```

## Keywords

`int`, `double`, `char`, `bool`, `if`, `else`, `switch`, `for`, `while`, `return`, `const`, `namespace`

## Operators

- Arithmetic: `+ - * / %`
- Relational: `== != < > <= >=`
- Logical: `&& || !`
- Assignment: `= += -= *= /=`

## Control Statements

- `if`
- `if-else`
- `switch`
- `break`
- `continue`
- `goto` only for learning

## Loops

- `for`
- `while`
- `do-while`

## Functions

```cpp
int add(int a, int b) {
    return a + b;
}
```

## Important Rules

- Initialize variables before use
- Use braces for clarity
- Prefer `const` for values that should not change
- Avoid `goto` in normal programs

## Best Practices

- Keep functions small
- Use meaningful names
- Test edge cases
- Read compiler errors carefully
