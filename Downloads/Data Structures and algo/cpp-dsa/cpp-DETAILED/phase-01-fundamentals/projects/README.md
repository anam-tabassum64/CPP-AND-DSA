# Phase 1 - Projects

## Beginner Projects

### 1. Calculator

- Problem Statement: Build a console calculator for basic arithmetic operations
- Features: add, subtract, multiply, divide, clear
- Concepts Used: variables, operators, functions, control flow
- Future Improvements: history, error handling, scientific operations

### 2. Student Marks Manager

- Problem Statement: Accept marks and display total, average, and grade
- Features: validation, grade mapping, summary output
- Concepts Used: input/output, conditions, loops, functions
- Future Improvements: file storage, class-based design

### 3. Number Utility Program

- Problem Statement: Perform common number checks and transformations
- Features: even/odd, prime, palindrome, digit sum
- Concepts Used: loops, conditions, functions
- Future Improvements: more number theory operations

### 4. Pattern Printer

- Problem Statement: Print common star and number patterns
- Features: menu-driven pattern selection
- Concepts Used: loops, nested loops, control flow
- Future Improvements: custom spacing and dynamic sizes

### 5. Command-Line Info Display

- Problem Statement: Read and display basic command-line arguments
- Features: argument count, argument listing
- Concepts Used: `main`, command-line arguments, output
- Future Improvements: option parsing

## Intermediate Projects

### 1. Menu-Based Utility Suite

- Problem Statement: Combine several beginner programs into one menu system
- Features: calculator, number tools, grading, pattern printing
- Concepts Used: functions, switch, loops, assertions
- Future Improvements: modular files, logging

### 2. Debugging Demo App

- Problem Statement: Create a small program that demonstrates debugging output and assertions
- Features: input validation, checks, diagnostic messages
- Concepts Used: assertions, control flow, basic debugging
- Future Improvements: structured logs
