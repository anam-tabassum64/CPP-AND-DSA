# Phase 1 - C++ Fundamentals

This phase builds the foundation for every later topic in C++, including DSA, competitive programming, object-oriented programming, and interview preparation.

It focuses on understanding how C++ works at the language level, how programs are compiled and executed, and how to write clean beginner-friendly code with confidence.

## Introduction

C++ is a general-purpose, compiled programming language designed for performance, control, and flexibility. It is widely used in:

- Data Structures and Algorithms
- Competitive Programming
- System Software
- Game Development
- Embedded Systems
- High-performance applications
- Technical interviews and coding assessments

This phase teaches the core language mechanics that every C++ developer must know before moving to advanced topics.

## Learning Objectives

- Understand what C++ is and where it is used
- Learn the compilation process from source code to executable
- Write basic C++ programs using variables, operators, conditions, loops, and functions
- Understand scope, lifetime, and storage classes
- Use input and output correctly
- Build confidence with debugging and basic program tracing

## Prerequisites

- Basic computer usage
- Familiarity with any programming language is helpful but not required
- Willingness to practice writing small programs

## Skills You Will Gain

- Reading and writing basic C++ programs
- Understanding compiler and linker behavior
- Using variables, constants, and data types correctly
- Applying control flow and loop logic
- Writing reusable functions
- Explaining basic C++ syntax in interviews
- Debugging simple compilation and logic errors

## Topics Covered

| Topic Group | Concepts |
| --- | --- |
| Introduction to C++ | History, features, applications |
| Compilation Process | Source file, preprocessor, compiler, assembler, linker, loader |
| Basic Syntax | Variables, data types, literals, keywords, identifiers |
| Core Language | Constants, namespaces, input/output, operators, expressions |
| Type System | Type conversion, implicit and explicit casting |
| Program Structure | Scope, lifetime, storage classes |
| Control Flow | `if`, `switch`, loops, `break`, `continue`, `goto` |
| Functions | Declaration, definition, prototypes, overloading, inline functions, default arguments, command-line arguments |
| Debugging | Basic debugging and assertions |

## Learning Path

1. Start with the introduction to C++ and the compilation process
2. Learn syntax essentials such as variables, data types, literals, and identifiers
3. Move to input/output, operators, and expressions
4. Study constants, namespaces, scope, lifetime, and storage classes
5. Learn control flow using `if`, `switch`, loops, `break`, and `continue`
6. Practice functions, prototypes, overloading, inline functions, and default arguments
7. Study command-line arguments, debugging, and assertions
8. Apply the concepts in examples, exercises, and mini projects

## Expected Outcomes

By the end of this phase, you should be able to:

- Read and explain a simple C++ program
- Write small programs independently
- Trace execution through conditions, loops, and functions
- Identify compilation errors more quickly
- Explain the basic role of the compiler, linker, and loader
- Prepare for the next phase without syntax hesitation
