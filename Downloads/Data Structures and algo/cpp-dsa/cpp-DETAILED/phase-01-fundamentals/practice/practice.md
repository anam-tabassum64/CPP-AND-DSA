# Phase 1 - Practice Strategy

## Learning Strategy

- Learn one topic group at a time
- Read the theory first
- Write at least one small program for every concept
- Rewrite the same program without looking at notes

## Practice Strategy

- Start with very small problems
- Focus on correctness before speed
- Check outputs for edge cases
- Repeat weak topics regularly

## Milestones

- I can solve basic syntax problems without help
- I can explain the compilation pipeline
- I can write small function-based programs
- I can debug beginner errors confidently
- I am ready for arrays and strings
