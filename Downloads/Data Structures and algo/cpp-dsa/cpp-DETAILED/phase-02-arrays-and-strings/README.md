# Phase 2 - Arrays and Strings

Phase 2 turns C++ syntax into practical problem solving. You will learn how to store data in contiguous memory, traverse it efficiently, and solve the most common beginner DSA problems using arrays, character arrays, and strings.

## Phase Overview

This phase focuses on:

- Arrays as the first structured data container
- 2D arrays for matrix-style problems
- Character arrays and C-style strings
- `std::string` for modern text handling
- Core array and string algorithms
- Pattern printing as a warm-up for loops and nested logic

## Learning Objectives

- Understand how arrays map to memory
- Learn traversal, search, update, insertion, and deletion patterns
- Use character arrays and `std::string` correctly
- Compare C-style strings with modern C++ strings
- Solve basic array and string algorithm problems
- Build confidence with nested loops through pattern printing

## Prerequisites

- Phase 1 fundamentals
- Variables, loops, functions, and basic I/O
- Comfort with compiling and running small C++ programs
- Basic understanding of time complexity

## Topics Covered

- Introduction to arrays
- 1D arrays
- 2D arrays
- Character arrays
- `std::string`
- String operations
- Array algorithms
- String algorithms
- Pattern problems
- Mini projects

## Skills You'll Gain

- Index-based thinking
- Memory and traversal intuition
- String processing fluency
- Problem decomposition
- Pattern recognition for DSA questions
- Cleaner input handling in C++

## Learning Outcomes

By the end of this phase, you should be able to:

- Declare, initialize, and traverse arrays
- Work with rows and columns in 2D arrays
- Read and manipulate character arrays safely
- Use `std::string` members and common utilities
- Solve linear search, reverse, rotate, frequency, and prefix sum problems
- Solve palindrome, anagram, substring, and tokenization problems
- Print common star, number, alphabet, hollow, pyramid, and diamond patterns
