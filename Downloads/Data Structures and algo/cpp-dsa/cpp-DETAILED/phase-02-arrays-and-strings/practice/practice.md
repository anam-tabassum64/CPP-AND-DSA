# Phase 2 - Practice Strategy

This phase works best when you move from concept to implementation quickly.

## Study Strategy

- Read one topic
- Solve 3 to 5 examples
- Complete 5 to 10 exercises
- Revisit complexity
- Repeat the pattern with the next topic

## Topic-wise Order

1. 1D arrays
2. 2D arrays
3. Character arrays
4. `std::string`
5. Array algorithms
6. String algorithms
7. Pattern problems

## Revision Plan

- Day 1: Arrays basics and traversal
- Day 2: Search, update, insertion, deletion
- Day 3: 2D arrays and matrix traversal
- Day 4: C strings and `std::string`
- Day 5: Array algorithms
- Day 6: String algorithms
- Day 7: Pattern printing and mixed revision
