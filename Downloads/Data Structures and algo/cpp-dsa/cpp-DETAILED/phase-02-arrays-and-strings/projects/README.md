# Phase 2 - Mini Projects

Each project should include:

- Objective
- Features
- Concepts Used
- Future Improvements

## Beginner Projects

### 1. Student Marks Analyzer

- Objective: Compute totals, averages, and grades for a small class
- Features: subject-wise input, total, average, highest/lowest marks
- Concepts Used: arrays, loops, conditionals, functions
- Future Improvements: file input, class-wise reports, ranking

### 2. Matrix Calculator

- Objective: Perform basic matrix operations
- Features: addition, subtraction, transpose, display
- Concepts Used: 2D arrays, nested loops, functions
- Future Improvements: multiplication, determinant, dynamic sizes

### 3. String Utility Toolkit

- Objective: Bundle common string operations into one menu
- Features: reverse, palindrome, anagram, frequency count
- Concepts Used: `std::string`, loops, conditionals
- Future Improvements: file-based input, punctuation handling

### 4. Text Statistics

- Objective: Analyze text and report simple metrics
- Features: character count, word count, sentence count
- Concepts Used: strings, `stringstream`, loops
- Future Improvements: reading from files, CSV support

### 5. Password Strength Checker

- Objective: Judge password quality using basic rules
- Features: length checks, character category checks, score output
- Concepts Used: strings, conditionals, frequency checks
- Future Improvements: blacklist checks, entropy scoring

## Intermediate Projects

### 6. Mini Spreadsheet

- Objective: Store and update tabular numeric data
- Features: row/column display, updates, sums, averages
- Concepts Used: 2D arrays, loops, functions
- Future Improvements: file storage, formulas, sorting

### 7. Contact Management System

- Objective: Manage simple contact records in memory
- Features: add, search, display, delete
- Concepts Used: arrays, strings, searching
- Future Improvements: file persistence, sorting, validation

### 8. Text Editor

- Objective: Implement small text transformations
- Features: reverse, case change, trim, search
- Concepts Used: strings, character processing, `stringstream`
- Future Improvements: multi-line editing, file save/load

### 9. CSV Analyzer

- Objective: Parse and summarize CSV-like text
- Features: row split, count fields, simple stats
- Concepts Used: strings, tokenization, arrays
- Future Improvements: quoted field handling, file I/O

### 10. Sudoku Board Validator

- Objective: Validate a 9x9 board layout
- Features: row checks, column checks, box checks
- Concepts Used: 2D arrays, loops, boolean logic
- Future Improvements: solver, input parser, UI
