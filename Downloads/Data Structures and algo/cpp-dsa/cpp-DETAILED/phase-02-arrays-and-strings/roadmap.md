# Phase 2 - Learning Roadmap

This roadmap gives the correct learning order for Arrays and Strings.

## Learning Sequence

| Step | Topic | Why It Comes Here |
| --- | --- | --- |
| 1 | Introduction to Arrays | Start with the mental model of contiguous storage |
| 2 | 1D Arrays | Learn the simplest array form first |
| 3 | Traversal | Understand how to read every element safely |
| 4 | Updating | Learn how to modify values by index |
| 5 | Searching | Solve the first complete array problem pattern |
| 6 | Insertion and Deletion | Understand data movement in fixed-size storage |
| 7 | Copying and Passing to Functions | Learn how arrays behave in parameters |
| 8 | 2D Arrays | Extend the same idea to matrices |
| 9 | Character Arrays | Move from numbers to text at the raw memory level |
| 10 | Null Character | Understand how C strings end |
| 11 | C Strings | Practice classic text handling |
| 12 | `std::string` | Switch to modern and safer string handling |
| 13 | String Operations | Learn constructors, capacity, modifiers, and iteration |
| 14 | Array Algorithms | Build common problem-solving patterns |
| 15 | String Algorithms | Apply the same logic to text problems |
| 16 | Pattern Problems | Strengthen nested loop thinking |
| 17 | Mini Projects | Combine concepts into small working programs |

## Recommended Study Flow

1. Read the array introduction and memory layout
2. Practice 1D array declaration, initialization, traversal, and update
3. Learn insertion, deletion, copying, searching, and passing arrays to functions
4. Study 2D arrays and matrix traversal
5. Move to character arrays and C strings
6. Learn `std::string` constructors, capacity, modifiers, iterators, and operations
7. Solve array algorithms like reverse, rotate, frequency, and prefix sum
8. Solve string algorithms like palindrome, anagram, tokenization, and pattern matching basics
9. Practice pattern printing
10. Complete the mini projects

## Milestones

- I can explain why arrays are contiguous
- I can trace index-based traversal without mistakes
- I can work with matrices and strings in C++
- I can choose between char arrays and `std::string`
- I can solve beginner DSA problems independently
- I am ready for pointers and references after this phase
