# Phase 2 - Coding Questions

## Easy

- Print all elements of an array
- Find the sum of array elements
- Find the maximum and minimum element
- Count even and odd numbers
- Reverse an array
- Check whether an array is sorted
- Search an element linearly
- Count the frequency of a number
- Reverse a string
- Check whether a string is a palindrome
- Count vowels and consonants
- Convert uppercase to lowercase

## Medium

- Left rotate an array by `k`
- Right rotate an array by `k`
- Remove duplicates from a sorted array
- Merge two sorted arrays
- Find the missing number
- Find the second largest element
- Calculate prefix sums
- Find the maximum subarray sum
- Check whether two strings are anagrams
- Count words in a sentence
- Extract a substring safely
- Tokenize a string using `stringstream`

## Hard

- Longest substring without repeating characters
- Minimum window style warm-up problems
- String pattern matching basics
- Matrix spiral traversal
- Transpose and reverse-based matrix operations
- Maximum sum submatrix warm-up

## Company-Wise Themes

- LeetCode: arrays, strings, prefix sums, two pointers
- GeeksforGeeks: matrix traversal, frequency, basic string logic
- HackerRank: input-heavy array and string problems
- CodeStudio: pattern problems and beginner DSA
- Codeforces: implementation-heavy simulation and edge cases
